Propósito del sitio


El proposito de esta pagina es explicar los atributos y valores como complementos de las etiquetas HTML para poder conseguir una pagina de calidad.


Atributos y valores


Los atributos y valores complementan las etiquetas aportando información necesaria para su correcto desempeño y así poder estar totalmente definidos,
por otra parte los valores introducen los datos necesarios para los atributos, como temas de colores o direcciones de enlaces, estos son algunos ejemplos de valores.



  • Id: el atributo Id permite identificar distintas etiquetas para poder diferenciarlas, por lo que no podrían exitir 2 etiquetas con el mismo Id.
  • Style: Este atributo viene de CSS, pero se utilza en la codificación de Html, esto nos sirve para poder agrar temas como color en la pagina.
